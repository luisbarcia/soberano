## Overview

A fully self-hosted documentation platform that serves as the single source of truth for internal and public-facing technical docs.

## Architecture

- **Hugo** with custom theme for consistent branding
- **Pagefind** for client-side full-text search
- **GitHub Actions** for automated build and deploy on merge
- **Markdown** with custom shortcodes for callouts, tabs, and API references

## Results

- 200+ pages of structured documentation
- Sub-second search across all content
- Automated deployment on every commit
